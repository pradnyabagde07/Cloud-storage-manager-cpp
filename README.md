# Cloud Storage Manager in C++

A simple menu-driven cloud storage simulation built with C++17.

### Features
- Sign up / Login
- Upload / Download files (simulated)
- Create folders
- Search files
- Version control
- Trash / Restore
- Share via token
- Quota management

### Build Instructions
```bash
cmake -S . -B build
cmake --build build
./build/cloud_storage
```
